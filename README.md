# The Last Mandate

The Last Mandate is a decision-driven city leadership simulation built with Python.

The player leads a fictional city through political, economic and social crises. Decisions can create immediate, hidden, delayed and conditional consequences.

## Current Status

The project is in early development.

## Planned Features

- Dynamic city statistics
- Decision-based events
- Immediate and delayed consequences
- Characters with relationships and memory
- Political factions
- Conditional and procedural events
- Multiple endings
- Save and load functionality
- Web API and browser interface
- Real-world deployment

## Technology

The initial version is being developed as a Python terminal application.

Future versions will include:

- FastAPI
- PostgreSQL
- Automated testing
- Web-based user interface
- Cloud deployment
